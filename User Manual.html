AAKASH BASED SMART CLASS APPLICATION

How to Use

Select Teacher OR Student

Then following options will be available

  1. PUBLIC DISCUSSION
    1. Click on Public Discussion Button
    2. Type the message required that is sent to every device on network
    3. Click on small arrow beside the typed message to send the message
  2. PRIVATE MESSAGE
    1. If Teacher then you will get Student list else vice-versa
    2. Click on the person to send private message
    3. Send message as described earlier
  3. WHITEBOARD
    1. If Teacher, then start moving your finger as you wish
    2. To Change colour or width press the options button on the top right corner
    3. Clear will reset the canvas
    4. P.S: Dont't forget to change the colour after using eraser
    5. If student then just observe because drawing will be drawn automatically
  4. FILES
    1. If teacher then select Send File
    2. Browse to the required file and select it
    3. File Size should not exceed 5 MB
    4. Then the received file can be found by the students in the Received Folder
  5. NOTE
    1. Restart the application if any unexpected action occurs
    2. The Received Files are automatically stored in /sdcard/SmartClass/
    3. It is not possible to message privately to fellow student
    4. It is possible to have more than one teachers
